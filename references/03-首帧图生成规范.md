# 阶段3参考：首帧图生成规范

> 待阶段2调稳后调试

## 工具选择

1. **即梦**（首选）- image2image 模式
2. **AIX GPT-img2**（即梦失效时备选）
3. **香蕉**（备选）

## 输入

- 阶段2输出的姿态参考图
- 阶段1输出的九宫格定妆照（角色一致性参考）

## 生成逻辑

图生图模式：
- 图片1（姿态参考图）：提供姿态/动作
- 图片2（九宫格定妆照）：提供角色特征

## 即梦图生图提示词结构（待验证）

```
参考图生图：请严格根据图片1中的角色，生成一张人物的六视图角色定妆照；
不要出现文字，要求纯白色背景，有人物面部正面特写、人物面部45度侧面特写、
人物面部背面特写、人物正面全身照、人物45度侧面全身照以及人物背面全身照。
该人物穿着图片2中的服饰。注意：只输出成一张图片。
```

## 一致性控制原则

- 即梦优先：即梦图生图一致性较好时使用
- 一致性失效：切换 AIX GPT-img2 或香蕉
