# 阶段3：首帧图生成

## 工具优先级

1. **GPT-img2**（首选）- AIX 平台
2. **香蕉（nano）**（次选）
3. 即梦（一致性控制不佳，阶段3不使用）

## 输入

- **图1**：阶段2输出的姿态参考图（姿态/构图来源）
- **图2**：阶段1输出的九宫格定妆照（角色特征来源）

## 核心提示词

与阶段2完全相同：

```
【图2】是一张角色九宫格定妆图，请用【图2】的角色，替换【图1】的角色，请分析【图2】提取出准确的角色特征，确保角色一致性；最终按照【图1】的构图生成图片，画幅比例1:1
```

## 工作原理

- 图1提供目标姿态/构图
- 图2提供角色外观特征
- 通过角色替换，生成"新角色摆出指定姿态"的首帧图

## 验证记录

- **2026-04-24**：AIX 平台 + GPT-img2 测试通过，角色一致性和姿态还原度均达标 ✅

## 阶段1+2+3 串联流程

```
阶段1：九宫格定妆照（角色特征）
阶段2：姿态参考图（姿态构图）
阶段3：首帧图（角色替换后的目标姿态图）
```
