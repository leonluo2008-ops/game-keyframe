# 阶段2：姿态参考图生成

## 工具优先级

1. **GPT-img2**（首选）- 效果最佳
2. **香蕉（nano）**（次选）
3. 即梦（效果不好，本阶段不使用）

## 输入

- **图1**：姿态参考图（构图来源）
- **图2**：九宫格定妆照（角色特征来源）

## 核心提示词

```
【图2】是一张角色九宫格定妆图，请用【图2】的角色，替换【图1】的角色，请分析【图2】提取出准确的角色特征，确保角色一致性；最终按照【图1】的构图生成图片，画幅比例1:1
```

## 姿态参考图要求

- 严格基于九宫格定妆照中的角色特征
- 单一姿态动作
- 像素风背景干净（纯白或透明）
- 角色全身出镜
- 动作清晰可辨识

## 工作原理

- 图1提供姿态/构图
- 图2提供角色外观特征
- 通过角色替换实现"同姿态、不同角色"

## 背面行走参考图的获取（重要教训）

**问题**：纯背部视角 + 俯视 + 四足行走 + 正对镜头远去 —— 这种组合的真实参考图极难找到，网络图片大多是侧面或斜后视角。

**解决方案**：用即梦 AI 生成参考图，而非依赖真实摄影。

### ✅ 已验证成功的背面行走参考图 Prompt

**用途**：生成"背部行走远离镜头 + 俯视"姿态参考图，用于工段2角色替换

**Prompt**：
```
A small orange and white tabby cat seen from directly behind, walking straight away from the camera. Pure back view, the cat faces directly away from viewer. Slight high-angle top-down perspective. We see the complete back, the two back legs stepping with visible paws on the ground, the tail pointing up, the head centered at top. The cat's body is centered and symmetrical from this exact rear view. Four legs clearly visible showing walking stride. Pixel art style, clean lines, white background.
```

**即梦参数**：`ratio=1:1, resolution_type=2k`

**关键要素**（必须包含，否则生成斜后方视角）：
1. `"Pure back view"` — 纯背部
2. `"walking straight away from the camera"` — 正对镜头远去
3. `"Slight high-angle top-down perspective"` — 高角度俯视
4. `"Two back legs stepping with visible paws"` — 后腿踏步
5. `"Four legs clearly visible showing walking stride"` — 四足行走
6. `"Pixel art style"` — 像素风

### 工段2完整操作流程

1. 用即梦生成上述 Prompt → 得到背部行走参考图
2. 该参考图作为 **图1（姿态参考）**
3. 九宫格定妆照作为 **图2（角色特征）**
4. GPT-img2 执行角色替换 → 输出太空猫背部行走首帧图
