# 阶段2：姿态参考图生成

## 工具优先级

1. **GPT-img2**（首选）- 效果最佳
2. **香蕉（nano）**（次选）
3. 即梦（效果不好，本阶段不使用）

## 输入

- **图1**：姿态参考图（构图来源）
- **图2**：九宫格定妆照（角色特征来源）

## 核心提示词

```
【图2】是一张角色九宫格定妆图，请用【图2】的角色，替换【图1】的角色，请分析【图2】提取出准确的角色特征，确保角色一致性；最终按照【图1】的构图生成图片，画幅比例1:1
```

## 姿态参考图要求

- 严格基于九宫格定妆照中的角色特征
- 单一姿态动作
- 像素风背景干净（纯白或透明）
- 角色全身出镜
- 动作清晰可辨识

## 工作原理

- 图1提供姿态/构图
- 图2提供角色外观特征
- 通过角色替换实现"同姿态、不同角色"
